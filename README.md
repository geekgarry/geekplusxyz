# geekplusxyz
一个分享与发现的网站
个人博客网站，使用Vue搭建，UI是使用bootstrap3，从传统的jquery的框架继承而来，慢慢的会使用Vue UI框架重写。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
